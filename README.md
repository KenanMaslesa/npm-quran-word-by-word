# Quran words
## Installation

```bash
npm i @kmaslesa/quran-word-by-word
```

## Usage
```typescript
const quranWords = require('@kmaslesa/quran-word-by-word');

quranWords.getAllQuranWords();
quranWords.getWordsByPage(page); //page = (1-604)
```
